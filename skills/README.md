# FleetMesh Skills

Repo-managed FleetMesh skills live here.

```text
skills/all/      shared by every ship
skills/romulus/  Romulus-only skills
skills/vulcan/   Vulcan-only skills
```

The Ansible install/update playbooks register these built-in skills with each
ship. Local one-off commands can still live under `/etc/fleetmesh/scripts`.
